シミュレーション用建物モデル

                                  株式会社エスキューブド 2008/2/29

■概要
OpenHRP用のOSL棟VRMLモデルです。

■ファイル
aist.main.wrl ... OpenHRP向けのVRMLファイルです。
*.wrl         ... 実際のVRMLファイルです。
README.txt    ... このファイルです。

■使用方法
1. 解凍
解凍してください。

$ tar zxvf osl-model.tar.gz

2. 読み込み
GrxUIのItemTreeViewにおいてmodelを右クリックし、aist.main.wrlを読み込んでください。

■備考
座標系の姿勢はOpenHRPの座標系に準拠し、位置は左下としています。
